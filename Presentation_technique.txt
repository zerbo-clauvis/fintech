                             Présentation technique

Le logiciel est organisé de façon simple :
        
  [ UTILISATEUR ]
        ↓
[ INTERFACE WEB ]
(Pages - Formulaires)
        ↓
[ - BACKEND ]
(Views + Logique métier)
        ↓
[ MODELS ]
(Traitement des données)
        ↓
[ BASE DE DONNÉES ]
        
                     
Les technologie utiliser pour le projet sont : 
Frontend:  Html - React - Js
Backend:  python - django - Api Rest
Base de donne: mysql
